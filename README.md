# taro-react-starter

开箱即用的基于`Taro3` + `Vue3` + `Pinia` + `TailwindCSS` + `TypeScript` 的模板

这个工程的搭建笔记可以[在这里查看](https://welives.github.io/blog/front-end/engineering/taro/create-vue.html)

## 使用

```sh
pnpm install
```

### 运行

```sh
pnpm dev:h5
pnpm dev:weapp
```
